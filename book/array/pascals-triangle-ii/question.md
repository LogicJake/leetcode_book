### 问题描述
Given a non-negative&nbsp;index **k**&nbsp;where **k** &le;&nbsp;33, return the **k**<sup>th</sup>&nbsp;index row of the Pascal&#39;s triangle.

Note that the row index starts from&nbsp;0.
