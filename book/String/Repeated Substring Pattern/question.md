## Repeated Substring Pattern  
### 问题描述
**Example 1:**<br />
<pre>
**Input:** "abab"

**Output:** True

**Explanation:** It's the substring "ab" twice.
</pre>


**Example 2:**<br />
<pre>
**Input:** "aba"

**Output:** False
</pre>


**Example 3:**<br />
<pre>
**Input:** "abcabcabcabc"

**Output:** True

**Explanation:** It's the substring "abc" four times. (And the substring "abcabc" twice.)
</pre>
