## Perfect Number  
### 问题描述
We define the Perfect Number is a **positive** integer that is equal to the sum of all its **positive** divisors except itself. 


**Example:**<br />
<pre>
**Input:** 28
**Output:** True
**Explanation:** 28 = 1 + 2 + 4 + 7 + 14
</pre>


**Note:**
The input number **n** will not exceed 100,000,000. (1e8)
